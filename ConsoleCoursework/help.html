Справка по программе "Управление банковскими счетами"

Данная программа предназначена для управления списком банковских счетов, хранящихся в файле accounts.csv. Позволяет просматривать, добавлять, удалять и искать аккаунты.

Основной функционал

  • Просмотр базы данных – отображает список всех банковских счетов в таблице.
  • Добавление аккаунта – позволяет внести новый банковский счет в базу.
  • Удаление аккаунта – удаляет выбранную запись по номеру в списке.
  • Поиск – выполняет поиск аккаунтов по имени, номеру телефона или банковскому счету.
  • Сохранение данных – вручную сохраняет изменения в файл.
  • Выход – завершает работу программы.

Горячие клавиши

  • ↑ / ↓ – перемещение по меню.
  • Enter – выбор пункта меню.
  • Esc – выход из поиска или возврат в меню.

Формат файла данных

Файл accounts.csv содержит данные в следующем формате:

Ф.И.О,Телефон,Дата,Домашний адрес,Номер карты,Банковский счет

Пример записи:

John Doe,+7(123)456-78-90,2025-03-06,1234 Elm St,1234567890,9876543210
